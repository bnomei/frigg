# SCIP Fixture Pack

These fixtures are synthetic, hand-authored JSON test artifacts for graph ingest
regression tests in this repository.

- Source: authored in-repo by Frigg contributors for deterministic tests.
- Licensing: same as repository license (MIT).
- Third-party code/data: none.

The payload shape matches the JSON parser used by `crates/graph` for SCIP-like
ingestion tests (documents, occurrences, symbols, relationships).
